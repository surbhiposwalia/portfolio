About Me

As a kid, I was always amazed with computers. I was in 3rd grade when i learned my first programming language LOGO, i was so inspired with how just some set of characters and numbers can make circles or rectangles.

I did my internship at Bharti Airtel Mumbai and worked on a project of SalesMan's guide using Java. After completing my bachelors, i joined Infosys as a system engineer. There I had an opportunity to work project of one of the biggest aircraft manufacturers, Boeing, analyzed the working of Boeing tools and came up with a re-engineering solutions. I really enjoyed working the project and thereby became passionate in developing a full stack application and become a software developer. To learn all about the full stack development, i joined coding bootcamp. Now I'm excited to put my skills to use and also develop them.
